# Pict-Section-Modal

> Modal dialogs, confirmations, tooltips, and toast notifications for Pict applications

- Promise-based confirm and double-confirm dialogs
- Custom modal windows with arbitrary content and buttons
- Toast notifications with auto-dismiss and stacking
- Simple and rich interactive tooltips
- Full CSS custom properties theming
